🚗 PathShare | #FantasticInnovators
Don't just travel. Travel together.

🌟 Inspiration
PathShare was born from a personal challenge: standing on a one-way route where buses are less, not at time or stuked in heavy traffic,but want to go home fast and rest needing a lift, but being held back by the social anxiety of asking a stranger to stop. In urban environments, thousands of vehicles travel at constant speeds with empty seats because there is no communication layer between the person on the curb and the driver.

This "Invisible Transport" crisis is most acute at midnight, when public buses stop and cabs are unavailable. PathShare bridges this gap by turning private vehicles into a community asset.

## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key
3. Run the app:
   `npm run dev`
